Knight's Tour

How do I use the site?

Using this site you are able to click on the board to decide where you would like the knight to start. Underneath the board there are several options.

  • The "<-" button will let you see the previous move
  • The "Finish Tour" button will complete the tour
  • The "->" button will move the knight to the next position

The flag icon represents the starting position, and the knight represents the current position.
If the tour cannot be complete, red crosses will appear where the knight did not visit.

What is a Knight's Tour?

The Knight's Tour is a popular chess problem revolving around the knight piece. The goal is to move the knight in such a way that it visits every square exactly one time.

This specific implementation of the Knight's Tour is based on "Warnsdorff's Rule". In essence, this means the next move the knight will take depends on how many unvisited cells can then be accessed; the more the better.
